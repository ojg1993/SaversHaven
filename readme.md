Saver's Haven, a Django Rest Framework(DRF) based backend API server project, serves as a second-hand trading platform. It facilitates connections between users to share pre-loved items, providing affordability for buyers and offering sellers a platform to declutter and earn.

Saven also empowers users to save money while reducing waste and promoting the reuse of resources.
By facilitating seamless transactions, real-time communication through live chat, and comprehensive item management features.

## Project Overview
### Tech Stack

- **Backend:** Django(DRF), PostgreSQL, Nginx, Postman
- **Deployment:** Docker, Github Actions, AWS Elastic Beanstalk
- **Version Control:** Git, Github
